# Cafe Aroma - Online Coffee Shop

**Cafe Aroma** is a static web page representing an online coffee shop.  
The project showcases a clean, modern layout for displaying coffee products, daily specials, a gallery, and contact information using HTML and CSS.

## Features
- Attractive homepage with a welcoming heading and tagline
- Navigation menu linking to Menu, Specials, Gallery, and Contact sections
- Video banner showcasing coffee preparation (autoplay muted)
- Coffee Menu section with images, product names, descriptions, and "order now" call-to-action
- Today's Specials section highlighting featured products
- Gallery section displaying coffee products visually
- Contact section with email and phone number for customer inquiries
- Responsive design with flex layout and consistent styling using CSS

## Technologies Used
- HTML5
- CSS3

## Project Purpose
This project was created to:
- Practice structuring web pages with HTML
- Style pages with CSS and organize layouts
- Create visually appealing content using images and video
- Build a professional-looking repository showcasing front-end skills
